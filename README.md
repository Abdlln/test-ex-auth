Необходимо реализовать приложение для управления заметками пользователя.

**Требования:**

- **Страница аутентификации:**
  - ○ аутентификация по логину/паролю
- **Ролевой доступ к ресурсам (user, admin)**
  - ○ user имеет права на управление заметками, не должен видеть и переходить на экран управления пользователями
  - ○ admin имеет все права
- **Управление заметками пользователя:**
  - ○ Страница списка заметок
  - ○ добавление заметки
  - ○ редактирование заметки
  - ○ удаление заметки
- **Управление пользователями:**
  - ○ Список пользователя
  - ○ Регистрация пользователя
  - ○ Активация пользователя
  - ○ Деактивация пользователя
  - ○ Смена роли пользователя
  - ○ Удаление пользователя
