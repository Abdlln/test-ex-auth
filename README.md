
Тестовое задание на управление заметками с ролевым доступом.
**Ссылка на сайт в GitHub Pages https://abdlln.github.io/test-ex-auth/**

**Ключевые решения:**
- Аутентификация через токен + защита роутов в vue-router
- Разделение прав: `admin` видит панель пользователей, `user` — только заметки
- Состояние: Pinia-модули с типизацией
- Вёрстка: БЭМ, адаптив

**Требования:**

- **Страница аутентификации:**
  - ○ аутентификация по логину/паролю
- **Ролевой доступ к ресурсам (user, admin)**
  - ○ user имеет права на управление заметками, не должен видеть и переходить на экран управления пользователями
  - ○ admin имеет все права
- **Управление заметками пользователя:**
  - ○ Страница списка заметок
  - ○ добавление заметки
  - ○ редактирование заметки
  - ○ удаление заметки
- **Управление пользователями:**
  - ○ Список пользователя
  - ○ Регистрация пользователя
  - ○ Активация пользователя
  - ○ Деактивация пользователя
  - ○ Смена роли пользователя
  - ○ Удаление пользователя
